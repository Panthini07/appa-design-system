# Appa — Brand Color System

The official color palette for **Appa**, a calming procrastination-productivity app.

- **Teal** = primary (carries most of the UI)
- **Pink** = accent (used sparingly)
- **Orange + Periwinkle** = semantic states
- Full **Light** and **Dark** mode tokens, plus the signature gradient

🔗 **Live palette:** https://panthini07.github.io/appa-design-system/
