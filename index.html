Appa — Brand Color System

Teal-led · pink accent · semantics built from the palette · light + dark
Hierarchy: teal = primary (most surface area) → pink = accent (sparingly). Orange & periwinkle carry the semantic states.
Every role uses the same token name in both modes — a true 1-to-1 light/dark swap. Each token has one light value and one dark value.

🌈 Signature gradient

teal #11D2CB → #8FE9E2 → #FFE3EF → #FF8FC2 → pink #FF4D97  (luminous bridge — no grey mud)
